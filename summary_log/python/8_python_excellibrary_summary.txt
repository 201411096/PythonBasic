xw.view
	ㄴ 엑셀로 보내기
	ㄴ 사용방법
		ㄴ xw.view(dataframe)
		ㄴ xw.view(dataframe, xw.sheets.active) # 활성화된 시트로 보냄
	ㄴ 사용예시_01
		data = [
		    ['apple','pear','mango'],
		    [100,150, 200],
		    [120, 160, 170]
		]
		xw.view(data)
